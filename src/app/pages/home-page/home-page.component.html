SdI Upload

In some exceptional cases, an incoming invoice or notification might not reach the CPI tenant. This tool enables you to manually upload these documents (which can be downloaded from the Tax Authorities' website) directly into your tenant. From this point on, the processing of this invoice or notification will be exactly the same as the others.

This is a support tool that should only be used in case of error or missing documents!

Do not use this tool for documents that have already been received successfully! This could create duplicate entries in your system.
